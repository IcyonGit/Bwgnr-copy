Welcome to Github

Intro

Github is a site bought by microsoft several years ago and has expanded massively. Github is a place for developers to store, host, and edot code. Github is such a great sight to use because:

  1. Main features are all free

  2. Github gives you your own domain name

  3. Most languages are supported

Github does have its dowsides

  1. Github cannot run languages that run directly on a terminal like java (note: it can still give syntax for the language)

  2. Github cannot run server side languages like php (note: it can still give syntax for the language)

  3. Eventually there is an upper limit on storage and runtime but it is a huge amount



Create account

  1. Go to github.com

  2. At the top right select create account and follow the steps

  3. Once finished go to the right and select create repository, for now skip and additions

  4. click "add file" and make a file with any name you want (some language must be called something specific to work like html) followed by .insert-laguage-here (index.html)
