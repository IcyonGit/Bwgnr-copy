Github Repositories

A github repository is a workspace that stores code files. You can have a nearly unlimited amount of files in a repository. These files only react to each other when specified, so you can put all your work in one repository. However, it is much better to split it accross repositories since you can have a lot.



Repositories have several main functions:

  1. code storage
  2. checklist
  3. product hosting
  4. website hosting
  5. syntax checker

Github repositories can be a very important tool.
