Pre and Post Increment

Problem 1

int a = 5;
int b = 0;
        
b = ++a + a++ + ++a + a++;
System.out.println(a + " " + b);

 

int a = 5;
int b = 0;


     1     2     3     4
b = ++a + a++ + ++a + a++;
System.out.println(a + " " + b);

Execution Results

int a = 5;
int b = 0;


     1     2     3     4
     6     6     8     8
b = ++a + a++ + ++a + a++;
System.out.println(a + " " + b);  // 9  28

The output would be 9 28

The value of a is calculated by summing the total number of increments (both pre and post) and adding the sum to 5.

The value of b is calculated as follows:

1 : step "1" variable a is incremented to 6 (pre-increment) and then a value of 6 is stored for future addition.
2 : step "2" a value of 6 is stored for future addition and then variable a is increment to 7 (post-increment).
3 : step "3" variable a is incremented to 8 (pre-increment) and then a value of 8 is stored for future addition.
4 : step "4" a value of 8 is stored for future addition and then variable a is increment to 9 (post-increment).
5 : Addition is executed from left to right with the values 6 + 6 + 8 + 8 which equals 28.
