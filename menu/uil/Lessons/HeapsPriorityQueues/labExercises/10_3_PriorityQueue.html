10.3 Priority Queue

Source Code: PriorityQueue

Description

A priority queue is an abstract data type that supports the following operations:

  • add an item to the queue with an associated priority
  • remove the item from the queue that has the highest priority
  • peek at the item with highest priority without removing it (optional)

There are many ways to implement a priority queue. One way is to keep a list of items in a linked list and search through the list for the highest priority item. This implementation takes O(1) time to insert an element, and O(n) time to perform the search for the highest priority item. There are many more efficient implementations available for implementing a priority queue including using a heap. With a heap both the process of adding an item to the queue and removing the highest priority item take O(log n).

Directions

Implement a HeapPriorityQueue class that implements the following interface.

public interface Queue
{
    // adds item to queue and assigns priority
    // the smaller the priority number the higher the priority.
    // (1 = highest priority, 2 = next highest, ...)
    public void add(Object obj, int priority);
    
    // view item with highest priority
    public Object peek();
    
    // remove item with highest priority
    public Object remove();
    
    // returns number of items in queue
    public int size();
    
    // returns true if queue is empty;
    // otherwise returns false
    public boolean isEmpty();
    
}

Items should be added to the heap using the following class:

public class HeapNode
{
   // instance variables
   private Object data;
   private int priority;

   // constructor
   public HeapNode(Object d, int p)
   {
      data = d;
      priority = p;
   }
   
   // accessor methods
   public Object getData()
   {
      return data;
   }
   
   public int getPriority()
   {
     return priority;
   }
   
   // mutator methods
   public void setData(Object d)
   {
      data = d;
   }
   
   public void setPriority(int p)
   {
      priority = p;
   }
}
