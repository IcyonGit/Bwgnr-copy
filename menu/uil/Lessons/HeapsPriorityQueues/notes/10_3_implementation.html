11.3 Heap Implementation

Heap Implementation

A heap is created by partially filling an array with nodes while maintaining the heap property. The two most important operations in building the heap are add and remove. The add operation is responsible for properly placing nodes in the heap. Duplicate nodes are placed below previously entered items. The remove operation typically deletes the topmost node in the heap and maintains the heap property.

Applications of Heaps

  1. PriorityQueue - a priority queue is a container class that allows items to be retrieved according to some specific priority level. The highest priority item is removed first.
  2. HeapSort - a fast sorting algorithm that runs at O(n log n) with no worst case of O(n2).
