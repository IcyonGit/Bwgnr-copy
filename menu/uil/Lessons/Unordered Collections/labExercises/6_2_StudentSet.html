7.2 StudentSet

Description

Write a class named Student that contains the following attributes: first name, last name, and current GPA. Write another class named School that creates Student objects from information entered from the keyboard and stores the objects in a TreeSet collection. The School class should be able to retrieve Student objects from the set in ascending order by the GPA. Since the Student objects will be stored in a TreeSet, the Student class needs to implement the Comparable interface.

Comparable Interface

The Comparable interface declares only one method named compareTo(Object obj). This method compares two objects A and B and returns one of three values. It returns a positive number if A is greater than B, it returns a negative number if A is less than B, and it returns zero if they are equal.

In your implementation of the compareTo method the ordering of Student objects should be determined by their GPAs.
