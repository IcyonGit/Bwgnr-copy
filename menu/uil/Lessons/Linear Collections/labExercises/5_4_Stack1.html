Description

Write a program that inputs a line of text and uses a stack to print the words within the line in reverse order. Use the String class's split method in your implementation.

String split method

A token is a substring of a string. The following example illustrates how the split method can be used to break up a string into its basic tokens: 

String sentence = "this is a test";
String[] result = sentence.split(" ");  // a space is used to determine the beginning and end of token

for (int x=0; x < result.length; x++)  
   System.out.println(result[x]);

prints the following output: 

this
is
a
test
