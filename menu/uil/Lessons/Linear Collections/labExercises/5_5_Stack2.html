6.5 Stack2

Description

Write a program that simulates the operation of stack by maintaining a list of integers that can be added(pushed) or removed(popped) from the stack. Use a menu system, like the sample output below, to allow the pushing and popping from the stack. The push operation should add a random number in the range of 1 to 100 to the stack. The pop operation should remove the top number from the stack. The stack should be printed after each push or pop operation.

Hint: Inorder to print the stack you will need to repeatedly call the stack's pop method. This process will eventually empty the stack. To keep from losing the values in the stack you will need to store them in another stack as you print them. After printing, copy the values from the second stack back into the first stack.
