6.8 Queue1

Description

Write a program that allows you to enter and delete integers from a list. All additions to the list must occur at the beginning of the list and all deletions must occur at the end of the list (FIFO). Use the LinkedList class which implements the Queue interface. The program needs to be menu driven like the sample output below.

Interfaces and Implementing Classes

interface java.util.List<E>

int size()  
boolean add(E obj) appends obj to end of list; returns true
void add(int index, E obj) inserts obj at position index and higher to the right (adds 1 tgo their indices) and adjusts size
E get(int index)  
E set(int index, E obj) replaces the element at position index with obj; returns the element formerly at the specified position
E remove(int index) removes element from position index, moving elements at position index + 1 and higher to the left (subtracts from their indices) and adjusts size; returns the element formerly at the specified position
Iterator <E> iterator()  
ListIterator <E> listIterator()  

interface java.util.Queue<E>

boolean add(E obj) enqueues obj at the end of the queue; returns true
E remove() dequeues and returns the element at the front of the queue
E peek() returns the element at the front of the queue; null if the queue is empty
boolean isEmpty()  

class java.util.LinkedList <E> implements java.util.List<E>, java.util.Queue<E>

void addFirst( E obj)
void addLast(E obj)
E getFirst()
E getLast()
E removeFirst()
E removeLast()
