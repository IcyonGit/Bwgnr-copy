6.2 LinkedList 2

Description

Write a program that creates a linked list from an unknown number of strings, then creates a second list containing a copy of the first list, but in reverse order. Output should consist of the original list and the reversed list. The input of strings should continue until the sentinel value -1 is entered by the user.
