6.3 Dictionary

Description

Write a dictionary program. The program should contain the following capabilities:

  • add words and their definitions into the dictionary
  • remove words from the dictionary
  • print a list of words in the dictionary
  • search for a word's definition and display it on the screen.

Implementation

Your implementation should include the following classes:

Entry class
Dictionary class
Manager class
Driver class

Entry Class

This class represents a single entry into the dictionary. The class should contain two attributes: a word and its definition.

This class should also include the following behaviors:

  • Three constructors - a default constructor, a constructor that sets both the word and definition attributes to values specified in its parameter list, and a constructor that sets only the word attributes to a value specified in its parameter list The definition should be set to its default value.
  • Accessor methods - need an accessor method for each instance variable.
  • Mutator methods - need a mutator method for each instance variable.
  • Override Object class's equals method. Override the equals method so that it compares the word attribute of the class with a String value given in the parameter list. If they are equal it should return true; otherwise it should return false.
    Example 
    public boolean equals(Object o)
{
   String arg = (String)o;
   return word.equals(arg);
}
  • Override toString method - returns a String representation of both the word and definition attributes.

Dictionary Class

This class is responsible for creating and maintaining the list of Entries. It should implement this list as a LinkedList of Entry objects. It should include the follow behaviors:

  • Default constructor - initializes the LinkedList.
  • Accessor methods
    getEntry- given an Entry object, it performs a linear search on the list of entries looking for a specified word. If the word is found in the list it returns its corresponding Entry object, otherwise it returns null.
    getWordList - this method creates and returns an ArrayList containing the word attribute of each Entry in the list. (It returns the list of words in the list).
  • Mutator methods
    addEntry - given an Entry object, this method adds the object to the LinkedList.
    removeEntry- given an Entry object, this method removes the object from the LinkedList.
  • Helper method
    linearSearch - given an Entry object, this method should perform a linear search through the list looking for the specified Entry object (identified by its word). This method should utilize an Iterator when traversing the list.

Manager class

This class is the "manager" for the program. It is responsible for displaying and managing the program's menu system and instaniating the Dictionary object.

This class should contain the following methods (behavior):

  • Default Constructor - the constructor should instaniate the Dictionary object and an Scanner object. It should also call the method MainMenu to start the menu system.
  • MainMenu method - this method is responsible for displaying the main menu for the program.
  • addWord method - prompts the user for a word and its definition, creates an Entry object, and then adds the entry to the dictionary.
  • removeWord method - prompts the user for a word, creates an Entry object, searches the dictionary for the specified word. If it is found, it removes it from the list. If the word is not in the dictionary, it should print the message "Word not in dictionary".
  • getDefinition method - prompots the user for a word, creates an Entry object, searches the dictionary for the specified word. If the word is found, it prints its definition. If the word is not found, it should print the message "Word not in dictionary".
  • viewWordList method - This method should display a list of all the words in the dictionary.

Driver Class

This method contains the programs main method. It initiates the program by instaniating a Manager object.
