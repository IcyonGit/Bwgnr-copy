Stack Samples

class java.util.Stack<E> (AP subset)

E push(E item) pushes item onto the top of the stack; returns item
E pop() removes and returns the element at the top of the stack
E peek() returns the element at the top of the stack
boolean isEmpty() throws an exception if the stack is empty

Example 1 - Creating a Stack Object

Stack<Integer> stack = new Stack<Integer> ();

Example 2 - Pushing items onto Stack

for(int i=0; i < 10; i++)
{
   stack.push(i);
}

Example 3 - Popping items from Stack

for(int i=0; i < 10; i++)
{
   int item = stack.pop(i);
   . . .
}

Example 4 - Traversing a Stack

for-each

while(!stack.isEmpty())
{
   System.out.print(stack.pop() + " ");
}
